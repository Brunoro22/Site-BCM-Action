# Tema WordPress BCM ACTION

Tema WordPress profissional para a empresa BCM ACTION - Serviços Especializados.

## Instalação

1. Faça upload da pasta `wordpress-theme` para `/wp-content/themes/`
2. Acesse o painel de administração do WordPress
3. Vá para Aparência → Temas
4. Clique em "Ativar" no tema BCM ACTION

## Estrutura do Tema

\`\`\`
wordpress-theme/
├── style.css              # Estilos principais
├── functions.php          # Funções do tema
├── header.php            # Cabeçalho
├── footer.php            # Rodapé
├── index.php             # Template principal
├── page.php              # Template de páginas
├── single.php            # Template de posts
├── css/
│   └── custom.css        # Estilos customizados
├── js/
│   └── main.js           # JavaScript
└── template-parts/       # Partes de template reutilizáveis
    ├── content.php
    └── content-none.php
\`\`\`

## Funcionalidades

- Design responsivo e moderno
- Menu mobile adaptável
- Suporte a logo customizada
- Widgets no rodapé
- Lazy loading de imagens
- Validação de formulário
- Integração com WhatsApp
- Suporte a posts e páginas
- Otimizado para SEO

## Cores Customizáveis

Edite as cores no arquivo `style.css`:
- Primary: #8b0000 (Vermelho escuro)
- Accent: #ff8c00 (Laranja)
- Background: #f9f9f9 (Cinza claro)
